프로그램 이름: Cookie Run Game

프로그램 설명:
- Cookie Run Game(한국어 패치: 국희런 게임)은 플레이어가 장애물을 피하고 가능한 한 오래 살아남는 게임입니다.
- 게임 도중 아이템을 획득하여 치트 모드를 활성화할 수 있으며, 일정 시간 동안 무적 상태가 됩니다.
- 15초 동안 생존하면 게임 클리어 화면이 표시됩니다.
- 게임 종료 후 F2 키를 눌러 게임을 재시작할 수 있습니다.

사용 방법:
- 화살표 위 키: 점프
- 화살표 아래 키: 슬라이드
- C 키: 치트 모드 활성화
- F1 키: 정보 화면 보기
- F2 키: 게임 재시작 (게임 종료 및 클리어 후)
- space 키: 게임 준비 및 시작

게임 시작 방법:
1. 프로그램을 실행하면 초기 화면이 나타납니다.
2. 스페이스 바를 눌러 게임을 시작할 준비를 합니다.
3. 스페이스 바를 떼면 게임이 시작됩니다.

게임 목표:
- 장애물을 피하면서 가능한 한 오래 살아남으세요.
- 점프 및 슬라이드를 통해 장애물을 피하고, 아이템을 획득하여 치트 모드를 활성화할 수 있습니다.

게임 클리어 조건:
- 15초 동안 생존하면 게임 클리어 화면이 표시됩니다.

치트 모드 사용 방법:
- C 키를 눌러 치트 모드를 활성화합니다.
- 치트 모드 활성화 시 5초 동안 무적 상태가 됩니다.

아이템:
- 주기적으로 8초마다 랜덤한 위치에 아이템이 생성됩니다.
- 아이템을 획득하면 점수가 증가하고 치트 모드가 활성화될 수 있습니다.

게임 종료 후:
- 게임 종료 후 F2 키를 눌러 게임을 재시작할 수 있습니다.
- F1 키를 눌러 언제든지 정보 화면을 볼 수 있습니다.
